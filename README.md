🛍️ Customer Shopping Behavior Analysis

 📖 Overview

Customer Shopping Behavior Analysis is an end-to-end data analytics project that explores customer purchasing patterns using **Python, MySQL, SQL, and Power BI**. The project transforms raw retail transaction data into meaningful business insights by performing data cleaning, exploratory data analysis (EDA), SQL-based business analysis, and interactive dashboard visualization.

The primary goal is to understand customer behavior, identify purchasing trends, evaluate the impact of discounts and subscriptions, and provide actionable recommendations that support data-driven business decisions.

 🚀 Features

* Data cleaning and preprocessing using Python
* Missing value handling and feature engineering
* MySQL database integration using SQLAlchemy and PyMySQL
* SQL queries to solve real-world business problems
* Interactive Power BI dashboard with KPIs and visualizations
* Business recommendations based on analytical insights

🛠️ Tech Stack

* Python
* Pandas
* NumPy
* MySQL
* SQLAlchemy
* PyMySQL
* SQL
* Power BI
* Jupyter Notebook

 📊 Project Workflow

1. Data Collection
2. Data Cleaning & Preprocessing
3. Exploratory Data Analysis (EDA)
4. MySQL Database Integration
5. SQL Business Analysis
6. Power BI Dashboard Development
7. Business Insights & Recommendations

🔍 Business Questions Solved

* Revenue analysis by gender
* High-spending customers using discounts
* Top-rated products
* Shipping type comparison
* Subscriber vs. non-subscriber spending
* Products with the highest discount usage
* Customer segmentation (New, Returning, Loyal)
* Top products in each category
* Repeat buyers and subscription analysis
* Revenue contribution by age group

 📈 Results

The project uncovers valuable customer purchasing trends, identifies high-value customer segments, evaluates promotional strategies, and provides interactive dashboards that help businesses make informed, data-driven decisions.
